Quick Start

Up and running in 30 seconds

Install globally
$ npm install -g secretguard
$ secretguard init          # sets up git hook + config
$ secretguard scan .        # scan current directory
macOS / Linux via Homebrew
$ brew install secretguard
$ secretguard init
One-line installer
$ curl -sSL https://get.secretguard.dev | sh
Add pre-commit hook to existing repo
$ secretguard hook install
✓ Pre-commit hook installed at .git/hooks/pre-commit
✓ SecretGuard will now scan staged files before every commit
